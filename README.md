# 🏎️ CarHub

## ℹ️ Description

A next.js app that showcases various cars along with their descriptions, images, specs and more

## 🧬 Live Site

https://carhub-1.vercel.app

## 🔮 Tech Stack

- Next.js
- React
- Typescript
- Tailwindcss
- Postcss

## 📦 Installation

```bash
npm install
```

## 🚀 Running the app

```bash
# development
npm run dev

# build
npm run build

# start
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
